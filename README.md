# Script Finder

Script Finder 是一个用户脚本（userscript），它可以帮助你在任何网站上查找和管理用户脚本。它提供了一种方便的方式来搜索和安装来自 Greasy Fork 的用户脚本。借助 Script Finder，你可以轻松地一键将自定义脚本添加到你喜爱的网站上，从而增强你的浏览体验。

<img height=6px width="100%" src="[https://media.chatgptautorefresh.com/images/separators/gradient-aqua.png?latest](https://media.chatgptautorefresh.com/images/separators/gradient-aqua.png?latest)">

## 功能

-   根据网站域名搜索用户脚本
-   查看每个脚本的详细信息，包括作者、描述、安装数量、版本和评分
-   **自动翻译**：支持将外语脚本信息一键翻译为中文
-   **持久化**：记录按钮位置，防止页面刷新后位置重置
-   一键安装用户脚本

<img height=6px width="100%" src="[https://media.chatgptautorefresh.com/images/separators/gradient-aqua.png?latest](https://media.chatgptautorefresh.com/images/separators/gradient-aqua.png?latest)">

## 使用方法

1. 在浏览器中安装用户脚本管理器，例如 Tampermonkey 或 Violentmonkey
2. 访问 [Greasy Fork](https://greasyfork.org/zh-CN/scripts/570429-script-finder-stable-persistent) 并安装脚本
3. 安装完成后，访问任何网页，右侧会出现一个 "Scripts" 按钮（初次加载会亮起 4 秒）
4. 点击按钮打开面板，即可查看适用于当前站点的脚本列表
5. 点击“安装”按钮即可直接跳转安装
6. 拖动按钮可以改变其位置，位置会被自动记录

### 修改说明：
1.  **GitHub 仓库**：全部更新为 `HHXXYY123/script-finder-plus`。
2.  **Greasy Fork**：ID 更新为你的 `570429`。
3.  **头像**：使用了 GitHub 默认的动态头像接口 `https://github.com/HHXXYY123.png`。
4.  **徽章（Badges）**：原版中有些徽章是基于原作者仓库里的特定 JSON 文件的（比如总安装数统计），那些链接在你的仓库没配置 Action 之前会失效。我帮你替换成了 **GreasyFork 实时动态徽章**，这样它们能直接显示你脚本的实时数据。
5.  **更新记录**：我帮你把最近咱们聊的这几次重大更新（翻译、位置记忆等）也写进去了，看起来更专业。
